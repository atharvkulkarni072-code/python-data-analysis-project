# 💧 Water Reminder Application (Python)

## 📌 Description
This is a simple Python-based desktop notification application that reminds users to drink water at regular intervals. It helps maintain proper hydration using system notifications.

## 🚀 Features
- Sends desktop notifications every hour
- Uses Python time module for scheduling
- Uses plyer library for system notifications
- Lightweight and easy to use

## 🛠️ Technologies Used
- Python (Core Python)
- plyer (for notifications)

## ⚙️ How It Works
The program runs in an infinite loop and sends a notification every 1 hour reminding the user to drink water.
